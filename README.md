# pond-water-image-enhancement
GAN-based pond water image enhancement framework using Residual U-Net, CBAM attention, and computer vision techniques.

## Convert frame PNGs into one file

Use this command to convert all frame `.png` images in one folder into a single `.mp4` file:

```bash
python frames_to_video.py --input /path/to/frames --output /path/to/output/video.mp4 --fps 30
```
